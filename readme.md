# font
## IBM Plex Mono 
- 2022.01.26 까지 가장 나에게 맞는 폰트
- {} 가 매우 잘 구분되며, 알파벳 모양이 눈에 딱딱 들어옴
- 한글과 알파벳의 사이즈가 규칙성이 있음
```
# 한글 다섯자와 알파벳 8자가 딱맞음
il1:|i17
"      "
한글다섯자

# 중간에 알파벳이나 공백 넣어도 문제없음
한a글 체결강
한글 다a섯자
al phabaet
8 char fit
```
